# Booking Calendar Project

A full-stack web application built using **Angular 14** (Frontend) and **Python Flask** (Backend).  
Users can view available event slots, apply preferences, and book events.

---

## Tech Stack

### Frontend
- Angular 14
- Angular Material
- TypeScript
- RxJS

### Backend
- Python Flask
- SQLite
